Autosave Backups

Open Admin

Autosave backups are automatic snapshots stored locally in your browser. They help you restore an older version if something gets changed by accident.

  • Version history: Admin shows recent autosaves (up to 12) and lets you restore any snapshot.
  • Create snapshot now: Make a fresh autosave before big edits.
  • Free up space: If storage fills up, Admin can clear autosave history.
  • Keep a file: Export Backup downloads a .json file you can move to another device or store in iCloud/Drive.

Tip: Export Backup before clearing autosaves or clearing all app data.
