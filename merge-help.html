Merge Backup Instructions

Merge backup brings customers + appointments into the current device without deleting what’s already there.

Desktop → Phone (Recommended)

  1. On your computer, open the calendar and click Export Backup.
  2. On iPhone/iPad, you can also use Share Backup (AirDrop) to send the file to another device.
  3. Move the downloaded .json file to your phone (email it to yourself, iCloud Drive/Google Drive, AirDrop, etc.).
  4. On your phone, open the calendar page.
  5. Open Menu → tap Merge backup.
  6. Select the backup file.
  7. You’ll see a summary of how many customers/appointments were added.

Conflicts (What wins?)

  1. Merging is done by internal id values, not by customer name.
  2. If the same customer/appointment exists on both devices (same id), the imported backup wins.
  3. If you created the “same” customer separately on two devices, they will usually have different ids, so both will be kept (you may see duplicates).

When to use Import (replace) instead

  1. Use Import backup (replace) when you want the other device to become an exact copy.
  2. Import replaces the current device’s customers/appointments with the file.
Tip: Export a backup first on the device you’re merging into, just in case you want to roll back.
