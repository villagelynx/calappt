How It Works

Quick instructions for scheduling, customers, and backups.

Back to Calendar

Scheduling

Basics

  • Tap a time slot to schedule.
  • Tap an appointment to edit or delete it.
  • Use Court / Location for reminders and exports.

Navigation

Day, Week, Agenda

  • Day: Swipe left/right (iPhone Safari) or use Previous Day / Next Day.
  • Week: Tap Week. Use Previous Week / Next Week.
  • Agenda: Tap Agenda to see the week in time order.
  • Today: Jumps back to the current day/week.

Customers

Profiles

  • Customers drawer: add/edit customer name, cell, email, and notes.
  • Customer List page: open Profile to see lesson history.
  • Customer List page: use Text / Email links to open your device’s Messages/Mail app (you still tap Send). Add a cell/email first to see these.
  • Import contacts: Open Menu → Import contacts to bring in a .vcf (vCard) or Google Contacts .csv file.
  • Inside an appointment, use Text confirmation to message the student (opens your Messages app).

Backups

Admin

  • Menu: Export / Import backups, export Google (.ics).
  • Admin: autosave history (versions), restore older versions, clear autosaves.
  • Autosave backups: After you make changes (add/edit/delete), CalAppt saves a version automatically on this device.
  • Versions kept: The most recent 12 autosave versions are kept (older ones drop off).
  • Restore: Open Menu → Admin, then tap Restore on a version.
  • Export Backup: Use this when switching devices or before clearing data (it downloads a file you can keep).
  • Merge Backup: Use this to bring customers + appointments onto another device without deleting what’s already there (imported backup wins on conflicts).
  • Merge help (step-by-step)
  • Turn off: You can disable autosave in Admin.
