Import / Export Instructions

This calendar saves data locally in your browser. You can keep using it without a database, then export a backup file whenever you want to move or protect your data.

Export Backup

  1. Click Export Backup in the top header.
  2. The app downloads a .json file with customers, tasks, appointments, the weather city, the visible week, and the left-panel mode.
  3. Save that file somewhere safe.

Import Backup

  1. Click Import backup in the top header.
  2. Choose a previously exported .json backup file.
  3. The app restores the saved data into the current browser.

Export To Google Calendar

  1. Go to the week you want to export.
  2. Click Export Google (.ics).
  3. Save the downloaded .ics file.
  4. Open Google Calendar, choose Other calendars -> Import, then select the downloaded file.

Move Local Data To Your Cell Phone

  1. On your computer, click Export Backup.
  2. Move the downloaded .json file to your phone using one of these simple methods:
    • Email it to yourself
    • Save it to iCloud Drive, Google Drive, Dropbox, or OneDrive
    • Use AirDrop if you are on Apple devices
    • Use a USB cable and file transfer
  3. Open the calendar app/page on your phone.
  4. Tap Import backup and choose that file.

Import Contacts From Your Phone

  1. Open Menu -> Import contacts.
  2. Choose a .vcf (vCard) file or a Google Contacts .csv.
  3. iPhone: open a contact in the Contacts app -> Share Contact -> Save to Files, then pick that file in CalAppt.
  4. Many contacts: export a vCard from iCloud/Google, save it to Files on your phone, then import it.
This app is local-first. Data stays in the browser until you export it. If you clear browser storage or move to a different device, import the backup file to restore your data.
